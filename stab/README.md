# 起動方法

手順

使用ツール

- swagger editor
- swagger ui

1. dockerを起動
1. swagger editorを起動
1. stab.yaml を貼り付ける
1. Generate Server →　node.js Serverで実行
1. zipファイルを解凍して、npm start
1. local host起動

## Docker起動

docker run -d -p 80:8080 swaggerapi/swagger-editor

## Exe

npm start

## Local Host

http://localhost:8080/docs
